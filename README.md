## Build Steps
- Set up your Google Maps API key in `~/.gradle/gradle.properties`:
```
ROUTES_MAPS_API_KEY = YOUR_API_KEY
```

- Set up your Google Directions API key in `~/.gradle/gradle.properties`:
```
ROUTES_DIRECTIONS_API_KEY = YOUR_API_KEY
```
